# ordering-menu
Ordering from a menu fast-food

Sandwich Ordering System
A Python program that simulates a food ordering system where customers can build their own meal and get a final total.
Features

Order a sandwich, beverage, and fries
Mega-size your fries for an extra $1.00
Add ketchup packets at $0.25 each
Combo discount of $1.00 off when you order all three items
Prints a full order summary with running total
Supports multiple orders in one session

Menu
Sandwiches: Chicken $5.25, Beef $6.25, Tofu $5.75
Beverages: Small $1.00, Medium $1.75, Large $2.25
Fries: Small $1.00, Medium $1.50, Large $2.00
Ketchup: $0.25 per packet
How to use

Run the program in Python
Answer yes or no to each menu item
Select your size or type when prompted
Program applies a $1.00 combo discount if you order all three items
Choose to keep ordering or quit to see your final total

Built with

Python 3
